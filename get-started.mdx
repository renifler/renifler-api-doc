# 🚀 Bienvenue sur l'API Renifler

L'API Renifler vous permet d'interagir avec notre plateforme pour :

- 🔍 Analyser les technologies utilisées sur un site web
- 🛡️ Vérifier les configurations DNS et SSL
- 👤 Gérer les comptes utilisateurs et les avatars
- 🤖 Générer des résumés technologiques avec l'IA
- 📈 Surveiller les performances des sites web

## 🔗 URL de base

```http
https://api.renifler.io/v1
```

Toutes les requêtes doivent être effectuées en HTTPS avec le préfixe `/v1`.

## 🔐 Authentification

Incluez votre clé API dans l'en-tête `X-API-Key` :

```http
X-API-Key: votre_clé_api
```

Obtenez une clé API via votre [dashboard Renifler](https://dashboard.renifler.io/api-keys).

<Note>
Vous pouvez tester l'API sans clé en utilisant l'endpoint `/ping` :
```bash
curl https://api.renifler.io/v1/ping
```
</Note>

## 🛠️ Configuration locale

<AccordionGroup>
  <Accordion icon="github" title="Cloner votre documentation">
    Votre documentation est hébergée sur GitHub. Pour la cloner localement :
    ```bash
    git clone https://github.com/votre-org/docs-renifler.git
    cd docs-renifler
    ```
  </Accordion>
  <Accordion icon="terminal" title="Prévisualiser localement">
    Installez le CLI Mintlify :
    ```bash
    npm install -g mintlify
    ```
    Puis lancez le serveur local :
    ```bash
    mintlify dev
    ```
    Cela ouvrira la documentation sur `http://localhost:3000`.
  </Accordion>
</AccordionGroup>

## 🚀 Déployer vos modifications

<AccordionGroup>
  <Accordion icon="rocket" title="Installer l'application GitHub">
    Installez l'application GitHub de Mintlify depuis votre [dashboard](https://dashboard.mintlify.com) pour automatiser les déploiements.
  </Accordion>
  <Accordion icon="git-commit" title="Pousser vos modifications">
    Une fois vos modifications effectuées :
    ```bash
    git add .
    git commit -m "Mise à jour de la documentation"
    git push origin main
    ```
    Vos changements seront automatiquement déployés.
  </Accordion>
</AccordionGroup>
